📋 Member ID System

Each approved alumni is automatically assigned an ID. You can manually edit these IDs in the Approved tab.

✅ Approving Registrations

  • Go to Pending tab
  • Click 👁 View to see all details
  • Click ✓ Approve to move to alumni list

✏️ Editing Profiles

  • Go to Approved tab
  • Click ✏️ Edit on any row
  • All fields are editable including Member ID and batch

📅 Events Management

  • Go to the Events tab
  • Click Add New Event to post an event
  • Events will instantly appear on the public events page
